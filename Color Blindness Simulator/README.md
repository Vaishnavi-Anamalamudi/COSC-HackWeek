# Color Blindness Simulator

A production-ready React + Vite accessibility tool for previewing uploaded images through common color vision deficiency simulations.

## Features

- Drag, click, or paste image uploads
- Normal, protanopia, deuteranopia, tritanopia, and achromatopsia simulation modes
- Canvas-based image processing with intensity blending
- Interactive slider, split views, zoom, pan, fullscreen, and magnifier
- RGB histograms before and after simulation
- PNG, JPEG, and PDF export
- Recent image and simulation history

## Scripts

```bash
npm install
npm run dev
npm run build
```
