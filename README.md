# ⚡ CSS Art Festival — Pikachu

An award-style CSS Art Festival submission featuring a fully hand-coded Pikachu illustration created using only HTML5 and CSS3.

No JavaScript.
No SVG.
No Canvas.
No Images.

Everything is constructed using semantic HTML elements, gradients, borders, shadows, clip-paths, transforms, pseudo-elements, and modern CSS techniques.

---

## Overview

This project demonstrates advanced CSS art techniques by recreating an expressive Pikachu-inspired character in a premium electric atmosphere.

The artwork includes:

- Head
- Body
- Arms
- Legs
- Tail
- Ears
- Eyes
- Nose
- Mouth
- Cheeks
- Highlights
- Shadows
- Animated electric environment
